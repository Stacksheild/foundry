# my-service-catalog-entry

Scaffolded by `foundry create --template service-catalog-entry`. Edit
`service.yaml` with your service's real metadata, then run:

```bash
npm start
```

Open http://localhost:3000 to see it rendered, or `curl
localhost:3000/api/service` for JSON. `src/yaml-lite.js` only understands flat
`key: value` pairs — swap it for the `yaml` package if you need nesting.
