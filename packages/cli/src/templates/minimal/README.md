# my-foundry-app

Scaffolded by `foundry create`. This is Foundry's one built-in starter
template — a static starting point, not an AI-generated app. The
plugin-manifest-based, multi-template scaffolding engine described in
[docs/development-plan.md](https://github.com/Stacksheild/foundry/blob/main/docs/development-plan.md)
(Phase 1) is not built yet.

## Run it

```bash
node src/index.js
```
